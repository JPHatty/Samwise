# Glossary

## Terms

### ToolForge
n8n-based agent framework for dynamic tool creation and execution.

### Intent
High-level user goal expressed as structured data (see intent-spec.schema.json).

### Run Record
Audit log entry for agent execution (see run-record.schema.json).

### Health Check
Service validation that confirms operational readiness.

### Export
Snapshot of system state for backup and recovery.

### Constraint
Hard requirement that must be satisfied (non-negotiable).

### Policy
Operational rule enforced by agent behavior.

## Abbreviations

- **MCP**: Model Context Protocol
- **TLS**: Transport Layer Security
- **API**: Application Programming Interface
- **CLI**: Command Line Interface
- **WSL**: Windows Subsystem for Linux
